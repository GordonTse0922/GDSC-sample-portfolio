# GDSC sample site

## Installation 

### MacOS (Intel)
```shell
#Get the source code
git clone https://github.com/GordonTse0922/GDSC_sample_portfolio
#Go to source directory
cd GDSC_sample_portfolio
# Install dependencies
npm install
```
### MacOS (M1)
```shell
#Get the source code
git clone https://github.com/GordonTse0922/GDSC_sample_portfolio
#Go to source directory
cd GDSC_sample_portfolio
# Install dependencies
yarn install
```
